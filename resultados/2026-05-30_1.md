# Iteração - melhoria do F1-Score (plano `docs/011-improve-prompt-v2.md`)

## Objetivo

Na iteração anterior (`2026-05-29_1.md`, commit `62b2555`) o prompt v2 ficou **REPROVADO** por uma única métrica: **F1-Score = 0.86** (precisa ≥ 0.90). As outras 4 já passavam. Esta iteração aplica o plano 011, que ataca exclusivamente o F1.

## Mudanças aplicadas (somente em `prompts/bug_to_user_story_v2.yml`)

1. **Tier MÉDIO reforçado** — além dos ≥5 critérios Gherkin, agora orienta seções condicionais conforme o relato: `Critérios Técnicos` (recomendações de **solução**), grupos extras de critérios (Admins / Prevenção / Acessibilidade), `Exemplo de Cálculo` (reproduzindo números do relato) e `Contexto Técnico/Segurança/Bug`.
2. **Guarda-corpo de precisão ajustado** — explicita que **reafirmar dados já presentes no relato não é invenção**, destravando o bloco `Exemplo de Cálculo`.
3. **Tier SIMPLES mantido enxuto** — regra anti-inflar: bug simples entrega só story + 5 critérios, sem seção técnica, mesmo citando números. Classificação CoT reforçada para tratar "contagem/número errado simples" como SIMPLES.
4. **Few-shot rebalanceados** — +1 SIMPLES com número, +1 MÉDIO com `Critérios Técnicos`, +1 MÉDIO com `Exemplo de Cálculo`. Tier COMPLEXO intocado.
